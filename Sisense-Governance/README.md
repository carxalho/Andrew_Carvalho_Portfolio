# Sisense Governance Dashboards

## Objective
Establish visibility and control over the BI estate within Sisense.

## Approach
- Partnered with Sisense Technical Account Executive to access backend data via API.
- Built 11 Jupyter notebooks using Python libraries (os, json, requests, pandas) to extract and transform metadata
- Created three governance dashboards:
  - Dashboard Analytics: Usage metrics, sharing details, engagement.
  - Data Model Insights: Size, build schedules, activity trends.
  - User Overview: Role distribution, activity levels, access groups.


## Tools Used
Python, Jupyter Ntebook, Sisense, SQL

## Impact
Enabled data-driven BI governance, leading to the removal of inactive assets, prioritization of high-impact dashboards, and improved departmental efficiency.
