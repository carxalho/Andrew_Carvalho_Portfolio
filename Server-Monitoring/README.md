# Server Monitoring & Job Analytics

## Objective
Monitor server health and ETL job performance to prevent outages and improve reliability.

## Approach
- Used SSMS metadata views to build a star-schema model with fact and dimension tables.
- Created dashboards to track database size, row/column counts, and growth trends.
- Developed job monitoring dashboards with alerts for failures and performance anomalies.
- Implemented triggers for proactive notifications on server capacity and job duration.

## Tools Used
SQL Server (SSMS), T-SQL, Windows Functions, Sisense

## Impact
Prevented server crashes, reduced job failures, and improved response times. Increased accountability and system reliability across the data infrastructure.
